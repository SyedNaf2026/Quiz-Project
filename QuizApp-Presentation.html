Overview
What is QuizApp?
🎯
Purpose
An online quiz platform where creators build quizzes and takers attempt them with real-time scoring.
👥
3 User Roles
Quiz Creators build quizzes. Quiz Takers attempt and compete on leaderboards. Group Managers create and manage groups, assign quizzes, and track group performance.
⚡
Key Highlights
Timed quizzes, 4 question types, difficulty levels, navigator panel, leaderboard and dark mode.

🔐 Auth Flow
Register → BCrypt hash → Store in DB → Login → Verify hash → Issue JWT → Navigate by role
📝 Quiz Flow
Creator makes quiz → Adds questions → Taker browses → Starts timer → Submits → Score saved
🏆 Result Flow
Submit answers → Calculate score → Save to DB → Update leaderboard → Show breakdown
Database
Database Structure
👤 Users
  • PK Id
  • FullName
  • Email
  • PasswordHash
  • Role
  • CreatedAt
📂 Categories
  • PK Id
  • Name
  • Description
📋 Quizzes
  • PK Id
  • Title, Description
  • FK CategoryId
  • FK CreatedBy
  • TimeLimit, Difficulty
  • IsActive
❓ Questions
  • PK Id
  • FK QuizId
  • QuestionText
  • QuestionType
✅ Options
  • PK Id
  • FK QuestionId
  • OptionText
  • IsCorrect
🏆 QuizResults
  • PK Id
  • FK UserId, QuizId
  • Score, TotalQuestions
  • CompletedAt
